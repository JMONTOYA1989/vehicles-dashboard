# 🚗 Vehicle Sales Dashboard

This project consists of an interactive web dashboard built with **Streamlit**, designed to explore vehicle listing data.

The application allows users to:

- Filter vehicles by price range
- Filter vehicles by odometer range
- Filter vehicles by vehicle type
- View key metrics such as:
  - Total listings
  - Average price
  - Average odometer
- Generate interactive visualizations:
  - Price distribution histogram
  - Price vs. odometer scatter plot
  - Vehicle type distribution
- View a dynamically filtered data table
